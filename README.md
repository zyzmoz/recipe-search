# Recipe Search

Project built to find your favorite dishes!

![Screenshot](https://raw.githubusercontent.com/zyzmoz/recipe-search/misc/screen.png)

## This project uses:
- React
- Redux
- Axios
- Parcel Bundler

## Commands

Run development server (port 8080):
```
  npm start
```

Run build (default folder 'build'):
```
  npm build
```
